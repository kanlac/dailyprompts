# 发起 DNS 请求

# Q
- DNS 端口号是多少？
- 为什么使用 UDP 而不是 TCP？
- 简述 DNS 请求的完整过程
- 如何手动进行域名解析？

# A
DNS 端口号是多少？——53。

为什么使用 UDP 而不是 TCP？
1. UDP 更快，不需要 3 次握手
2. DNS 请求的数据量很小，适合放到 UDP 数据报里发送
3. 尽管 UDP 本身不可靠，但我们可以在应用层为其实现可靠性，比如超时处理
